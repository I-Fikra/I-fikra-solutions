# I-fikra-solutions

A configurable platform builder that enables organizations to create fully customized web applications through branding, theme customization, feature selection, and modular architecture.

---

### 🛠️ Tech Stack
- Angular
- PrimeNG
- TypeScript
- Tailwind and SCSS
- Transloco for localization

---

### ⚙️ Getting Started

```bash
git clone <your-repo-link>
cd <project-folder>
npm install
ng serve
```

Then open: **http://localhost:4200/**
